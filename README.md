## Description

**MahiDevStudio** is a professional portfolio website showcasing web development projects, skills, and services. It serves as a digital presence for demonstrating expertise in web development, including project portfolios, technical skills, and contact information.

### Key Points:
- Clean and modern user interface
- Responsive design for all devices
- Showcase of development projects
- Professional branding
- Contact and inquiry management

## User Manual

### Getting Started

#### 1. **Viewing the Website**
   - Open your web browser
   - Navigate to: `https://mahimuhtasim.github.io/mahidevstudio`
   - The homepage will load with all sections available

#### 2. **Navigation**
   - Use the **navigation menu** at the top to jump between sections:
     - Home
     - About
     - Portfolio
     - Skills
     - Contact

#### 3. **Exploring Sections**

   **Home Section:**
   - Introduction and headline
   - Quick overview of services

   **About Section:**
   - Background information
   - Professional summary
   - Qualifications

   **Portfolio Section:**
   - View completed projects
   - Click on projects for more details
   - Links to live demos and source code

   **Skills Section:**
   - Technical skills and proficiencies
   - Programming languages
   - Tools and frameworks

   **Contact Section:**
   - Contact form for inquiries
   - Email address
   - Social media links

#### 4. **Contact Form Usage**
   1. Scroll to the Contact section
   2. Fill in your details:
      - Name
      - Email
      - Subject
      - Message
   3. Click "Send Message"
   4. You'll receive confirmation

#### 5. **Mobile Experience**
   - Hamburger menu for small screens
   - Touch-friendly buttons
   - Optimized layouts for readability

## Installation

### Local Development Setup

```bash
# Clone the repository
git clone https://github.com/MahiMuhtasim/mahidevstudio.git

# Navigate to the project directory
cd mahidevstudio

# If using a local server (recommended)
# Python 3.x
python -m http.server 8000

# OR using Python 2.x
python -m SimpleHTTPServer 8000

# Open your browser and visit:
# http://localhost:8000
```

### No Build Process Required
This is a static website - just open `index.html` in your browser directly!

## Usage

### For Visitors:
1. Browse the portfolio and learn about the developer
2. Check out completed projects
3. Review technical skills
4. Get in touch via the contact form

### For Developers (Customization):

#### Editing Content:
- Edit HTML files in the main directory
- Update text, images, and links
- Modify CSS in the `styles.css` file for styling

#### Adding Projects:
1. Open `index.html`
2. Locate the portfolio section
3. Add new project HTML block
4. Update portfolio data

#### Changing Styling:
- Edit CSS files in the `css/` folder
- Modify colors, fonts, and layouts
- Test on different screen sizes
